# Камень-Ножницы-Бумага Online

Онлайн карточная игра "Камень-Ножницы-Бумага" для двух игроков.

## Правила игры

- Каждый игрок получает 6 случайных карт из 9 (по 3 каждого типа: камень/ножницы/бумага)
- Игроки расставляют карты в желаемом порядке
- 6 раундов с одновременным вскрытием карт
- Свап: можно менять соседние карты местами (до 3 раз за игру, не более 1 раза за раунд)

## Запуск

### Локально

```bash
npm install
npm start
```

Игра будет доступна на http://localhost:3000

### Docker

```bash
docker-compose up --build
```

## Технологии

- **Сервер**: Node.js, Express, Socket.IO
- **Клиент**: HTML5, CSS3, Vanilla JavaScript
- **Инфраструктура**: Docker, Docker Compose
